# 🧠 ALPHA SOLVE — STEM Vision

An interactive STEM platform that solves, explains, and visualizes problems.

## 🚀 Features
- Step-by-step math solver
- Function graphing
- OCR equation recognition
- AI-style explanation system
- Quiz system

## ⚙️ Tech Stack
- React (Vite)
- Chart.js
- Math.js
- Tesseract.js

```bash
